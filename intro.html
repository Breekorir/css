CSS- Cascading styles sheets

css is a programming language used to improve the presentation of a html page

css will help modify aspects like text size,colors background ,image sizes andshapes borders positions responsiveness(different looks on different screensizes) among other aspects of presentation

CSS is used to design user interphases for web page and it helps enhance user experience

CSS syntax(how do we write css)

a selector is used to target an element or a group of elements for styling

{}a declaration block -groups properties and values to be applied on the selector

property -a presentation aspect youd want to modifye.g color,width

value-the final look yuo are interested in for the page(red,fuulpage,400px)
